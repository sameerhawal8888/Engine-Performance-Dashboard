# Engine Oil Condition Detection Dashboard

## 📌 Overview
Operational health tracker classifying engine 
oil status into Healthy, Warning, and Critical 
categories via threshold-based ETL logic.

## 🛠️ Tools Used
- Power BI
- ETL Pipeline Design
- Statistics & Data Modeling

## 📊 Key Features
- 3-category oil health classification
- Threshold-based ETL logic
- Trend-based degradation visualization
- Proactive maintenance scheduling support
- Non-technical user friendly design

## 🔍 Key Features Built
- ETL threshold classification logic
- Degradation trend signals
- Maintenance cycle prediction
- Visual health status indicators
